# AmneziaWG Telegram Bot

Управление VPN-сервером через Telegram-бота с использованием AmneziaWG (обфускация трафика).

## Быстрый старт

\`\`\`
git clone https://github.com/SergeyIlins/amneziawg-bot.git
cd amneziawg-bot
chmod +x install.sh
sudo ./install.sh
\`\`\`

Следуйте инструкциям, введите данные бота.

После установки управляйте ботом командами /menu.
