# AmneziaWG Telegram Bot

Управление VPN-сервером AmneziaWG через Telegram-бота.
Установка в одну команду, полная автоматизация, обфускация трафика для обхода блокировок.

## ✨ Возможности

- 🤖 Управление через Telegram-бота (добавление/удаление клиентов, список, статистика).
- 🔐 Выдача клиентских конфигов с **полными параметрами обфускации** (AmneziaWG 2.0).
- 📱 Генерация `.conf` и QR-кодов для быстрого подключения.
- 🧹 Автоочистка просроченных клиентов (ежедневный таймер).
- ⚡ Автоматический перезапуск интерфейса при добавлении/удалении клиентов.
- 🛡️ Полный туннель (0.0.0.0/0) с возможностью смены параметров обфускации.

## 📋 Требования к серверу

- Ubuntu 22.04 / 24.04 или Debian 11/12.
- 1 vCPU, 512 МБ RAM, 5 ГБ диска.
- Статический публичный IPv4.
- Открытый UDP-порт (по умолчанию 587) и TCP-порт 8000 (для API).
